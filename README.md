# TheInternAcademy-Webdev-Task1
Create a website that can act as an image holder for users. Users can save images in them with different albums. Make a homepage, about page, contact page, signup page, login page. One can add images in different albums of their choice. Make the view page of images as much beautiful as you can. Along with it, one can add description to an image. Arrange the images in such a sequence in which they are mostly liked and mostly commented.
